# AI-Driven HealthCare Assistant

An AI-powered healthcare assistant built using Flask that provides users with disease awareness and health-related guidance through a simple web interface.

## Features

- AI-powered healthcare assistance
- Disease awareness interface
- User-friendly web application
- Flask backend
- HTML templates

## Tech Stack

- Python
- Flask
- HTML

## Project Structure

AI-Driven-HealthCare-Assistant/
│── app.py
│── requirements.txt
│── README.md
│── .gitignore
└── templates/
    ├── index.html
    └── start.html

## Installation

```bash
pip install -r requirements.txt
python app.py
```
